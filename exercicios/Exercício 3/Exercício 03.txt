    Qual é a importância de apresentar sua empresa utilizando a Internet?
        R: A internet gera uma visibilidade gigantesca de modo que possibilita as empresas a estarem dentro da casa das pessoas, que se beneficiam dessa área para expandir seus negócios

        R: É importante que tudo esteja organizado e atrativo, desde o design do site, a empresa deve ter uma identidade visual já estabelecida e expressa, além de departamentos de fácil acesso, agrupando os temas de acordo com seu relacionamento, expor apenas. não há poluição visual, e esclarece os passos que um cliente precisa seguir para realizar uma compra, alguns aspectos precisam ser considerados que irão mudar.
3.- Qual a diferença da Web 3.0 para as Web anteriores?
    R: A principal diferença da Web 3.0 é que ela consolida a mobilidade, mas ao reduzir o uso de intermediários para interação, os usuários têm maior controle sobre a privacidade e menos controle por grupos ou autoridades.
4.- Quais as vantagens e desvantagens os padrões da Web definidos pela W3C?

    Desvantagens: Implementações discordantes, Aumento da complexidade inicial;

8.- Sobre CSS3 (responda com exemplos):
    O que é?
    R: CSS3 é a teceira versão do CSS, uma linguagem de estilização do html que significa Cascading Style Sheet.
    Como escrever um estilo para um ID?
    R: #valor_id { propriedades de estilo }
    Como escrever um estilo para uma classe?
    R: .valor_classe { propriedades de estilo }
    Como escrever um estilo para uma tag dentro de uma classe?
    R: .valor_classe valor_tag { propriedades de estilo }
    Como escrever um estilo para uma tag?
    R: valor_tag { propriedades de estilo }
    O que é Box Model?
    R: Cada componente na página é definido por boxes retangulares, no css, esses boxes retangulares são descritos usando o box model padrão. Este modelo descreve o conteúdo do espaço ocupado por um elemento. Cada box possui 4 edges: margin edge, border edge, padding edge e content edge.
    O que é Floating?
    R: Floating é o ato de flutuar um elemento, a propriedade float do CSS determina que um elemento deve ser retirado do seu fluxo normal e colocado ao longo do lado direito ou esquerdo do seu containêr, onde textos  e elementos em linha irão se posicionar ao seu redor.
    O que o Floating muda do desenvolvimento Web atual em relação ao modo antigo (com uso de tabelas na construção do site)?
    R:O Floating facilita o posicionamento dos componentes ao longo da tela.
